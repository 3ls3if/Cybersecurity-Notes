# Custom Wordlist

## Crunch

```
# Run
crunch

# min length 1, max length 3, only numbers 1 2 3
crunch 1 3 123

# Saving the ouput
crunch 1 3 ab2 > wordlist.txt
```

## html2dic

```
# Run
html2dic <sample.html>
```

## Cewl

```
# Help
cewl -h

# Run
cewl <domain or IP address>

# Save the output
cewl <domain or IP address> -w sample.txt
```
